Purpose:

- The purpose of this web application is to transcribe .mp3 files into downloadable text files.


What makes this different:

- This app integrates Google Drive so you can share and edit the results with others on Google Docs.

- This app deletes the uploaded files immediately after they've been transcribed to protect user privacy.


How it Works:

- An .mp3 file is uploaded on HTML.

- When the "Transcribe" button is pressed, HTML sends a POST request to Python Flask backend.

- Flask uploads a copy of the file to an internal, temporary folder.

- Flask calls OpenAI's Whisper API to transcribe the uploaded file.

- After the transcribed texts are returned, the uploaded file is deleted immediately from the folder.

- If user refreshes while transcribing, the file is still deleted.

- Flask redirects you to the results page and you will have options to download to PC and download to Google Drive, if you sigend in.
